# 系统架构

## 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                     用户交互层（Streamlit）               │
│  搜索问答 | 题库练习 | 知识图谱可视化 | 掌握度仪表盘        │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                    AI推理层（LangGraph）                  │
│  ┌─────────────┐ ┌──────────────┐ ┌────────────────┐   │
│  │  检索Agent   │ │   推理Agent   │ │    出题Agent   │   │
│  └─────────────┘ └──────────────┘ └────────────────┘   │
│  ┌─────────────┐                                       │
│  │  复盘Agent   │                                       │
│  └─────────────┘                                       │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                   知识管理层                              │
│  ┌──────────────────┐    ┌────────────────────────┐    │
│  │   ChromaDB向量库   │    │   NetworkX知识图谱      │    │
│  │  (语义相似度检索)   │    │  (概念关系推理)         │    │
│  └──────────────────┘    └────────────────────────┘    │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                   文档处理层                              │
│  PDF解析 → 分块 → Embedding → 入库                       │
│  实体识别 → 关系抽取 → 图谱节点/边                        │
└─────────────────────────────────────────────────────────┘
```

## Agent协作流程

```
用户提问
    │
    ▼
[检索Agent] ──→ 向量库检索 Top-K 片段
    │              知识图谱路径查询
    │
    ▼
[推理Agent] ← 汇总检索结果
    │  多跳关系推理
    │  综合多来源答案
    │
    ▼
[出题Agent] ← 根据遗忘曲线选择薄弱知识点
    │  生成新题型
    │
    ▼
[复盘Agent] ← 用户答题结果
    │  更新知识点权重
    │  更新掌握度
    ▼
返回用户
```

## 数据存储

- `data/vectorstore/` — ChromaDB 持久化目录
- `data/graph/` — NetworkX 图谱序列化文件
- `data/uploads/` — 用户上传的PDF/ TXT原始文件
- `data/cache/` — 临时缓存

## 技术选型理由

| 组件 | 选型 | 理由 |
|------|------|------|
| LLM推理 | Ollama | 本地运行，隐私安全，支持qwen3/llama3 |
| 向量库 | ChromaDB | 轻量，Python原生，元数据过滤强大 |
| 图谱 | NetworkX | 简化版Neo4j，无需额外安装 |
| Agent | LangGraph | 状态机驱动，多Agent协作成熟方案 |
| 前端 | Streamlit | Python直出Web，开发效率极高 |
